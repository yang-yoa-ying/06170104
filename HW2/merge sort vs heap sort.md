# merge sort vs heap sort
## 耗時
merge sort = heap sort = NlogN(N筆資料)
## 原理
* heap sort -> complete binary tree
* merge sort-> Divide and Conquer 
## 穩定度
heap sort比merge sort不穩定
## 速度
理論上heap sort比merge sort快因為heap sort是in-place sorting(不須額外資源)
